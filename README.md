This is an opinionated gaming-oriented bootc image, an "immutable" linux-based operating system. Featuring the Cachy kernel, [Xenia the fox](https://xenia.efi.pages.gay/) in theming, and various other software tweaks and flatpaks that suit my, Xenia Meraki, tastes. Its emphasis is on gaming performance on


It can be used by first installing [Bluefin](https://projectbluefin.io/) (or any related Bootc image) and rebasing with command:


`sudo bootc switch ghcr.io/xeniameraki/xeniaos`


## Featuressss


-Linux/Arch Bootc


-Pre-setup Arch/AUR packages and applications, including Steam and Vesktop


-Niri, a tiling window manager


-Noctalia/Quickshell


-CachyOS Kernel


-Content creator oriented flatpaks set preinstalled


The project is in its infancy, I would not use this for any serious work/use for right now.
